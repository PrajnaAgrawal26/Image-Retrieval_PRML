Abstract

This project presents a suite of machine learning models for image classification and retrieval using both supervised and unsupervised learning approaches. The Vortex model employs a ResNet-32 architecture with residual blocks to mitigate vanishing gradients, enabling accurate classification of images into 10 categories. Additional models utilize deep feature extraction through ResNet50, followed by Linear Discriminant Analysis (LDA) for dimensionality reduction and class-discriminative enhancement. Aether leverages multinomial logistic regression with softmax-based probability estimation for multi-class classification, while Nuvora implements a Random Forest classifier to ensure robust prediction via ensemble learning. In contrast, Orion utilizes unsupervised K-Means clustering to group data based on feature similarity, mapping clusters to labels using majority voting. Together, these models form a comprehensive system for extracting, learning, and predicting complex visual patterns with strong generalization performance.
